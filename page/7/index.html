tldr——快速使用Linux命令

发表于 2018-09-13 分类于 Linux

tldr命令是一个Linux的帮助程序，简单理解就是man命令的精简版，非常适合轻度Linux用户，可以非常直观地展示一个命令的用法，比如解压命令tar的参数非常繁多，针对不同的打包文件有不同的参数，这时候tldr的威力就来了。

简介及安装

Simplified and community-driven man pages : http://tldr-pages.github.io/

GitHub地址：https://github.com/tldr-pages/tldr

很多时候，我们学习一个命令，仅仅想快速了解怎样使用，而不是了解它的全部用法或者阅读文档然后总结，显得很是累赘，著名的二八规则也可以套上，我们只用到一个命令的20%的功能就可以解决我们80%的需求。
因此想要快速了解一个命令的用法，tldr命令是个不错的选择，非常适合偶尔折腾一下Linux的朋友。

发表于 2018-09-13 分类于 Linux

简介

You-Get is a tiny command-line utility to download media contents (videos, audios, images) from the Web, in case there is no other handy way to do it.
如果没有其他便捷的方法，You-Get是一个微型命令行实用程序，可从Web下载媒体内容（视频，音频，图像）。

github地址：https://github.com/soimort/you-get

Ubuntu推荐使用pip安装：
