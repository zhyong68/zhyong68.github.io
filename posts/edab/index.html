安装manjaro系统

发表于 2018-09-21 分类于 Manjaro

安装Manjaro

下载

官方网站：https://manjaro.org/get-manjaro/
目前官方最新版本为17.1.12，支持都包括xfce,kde,gnome三种桌面环境，选择喜欢的下载，我安装的是kde桌面。

制作U盘启动

使用Rufus以DD方式写入到U盘，注意此操作会擦除U盘所有数据，请做好备份。
Rufus官方下载：http://rufus.akeo.ie/downloads/rufus-3.3p.exe

安装

进入Manjaro Live系统后，点击Install Manjaro即可进行安装，图形化界面非常方便，系统安装时可选择分区的方式：抹除磁盘、手动调整、与其它系统共的方式，可谓是非常方便。与其它系统共存的方法安装，可以方便使用Windows+Manjaro双系统，安装后自动加入Windows启动项。

进入boot时，记得将driver选择nonfree!!!

安装Yaourt

简介

Yaourt 是一个由 Julien Mischkowitz 所编写的 Bash 脚本，它是将 Pacman 与 AUR 这两者相结合的绝佳工具。通过 Yaourt 安装 AUR 中的包十分方便，它不仅简化了上述繁琐的过程，而且把这一过程半自动化，使用者只需在它的交互模式中简单的回答几个问题即可。此外，Yaourt 支持将结果以鲜亮的颜色输出，非常抢眼。

安装：sudo pacman -S yaourt

通过yaourt command即可列出所有的需要查询的命令或者软件，然后选择安装。

安装zsh

安装

  • 安装zsh: sudo pacman -S zsh
  • 安装oh-my-zsh 运行该脚本：sh install.sh
  • 修改当前用户(不需要sudo)默认shell：chsh -s /bin/zsh

配置

  1. 编辑~/.zshrc,修改zsh主题为ys
  2. 安装插件
  • 代码提示插件：zsh-autosuggestions
  • 代码高亮插件：zsh-syntax-highlighting
    将插件使用git工具clone到~/.oh-my-zsh/custom/plugins,编辑配置文件.zshrc,添加到插件中。 还可以添加extract,z,fzf等插件。

还可以使用网上的一个zsh主题：zeta-zsh,下载到oh-my-zsh/custom/themes启用即可（启用之后刷新配置）！

安装vim

安装：sudo pacman -S vim
配置

  1. 安装vimplug: curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
  2. 运行命令：wget https://github.com/stiverick610/vimrc/raw/master/vimrc -o ~/.vimrc,进入vim，运行：<leader> <leader> i进行安装插件!
    备注：
  3. 需安装Droid Sans Mono Nerd Font Complete字体，参考：https://www.zhyong.cn/posts/6daa/ ，然后修改终端字体即可。
  4. 手动编辑.vimrc文件，选择需要的插件（YCM选项等）,若安装YouCompleteMe插件报错，请查看官方文档(需安装cmake等)。
