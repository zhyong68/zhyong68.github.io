自用Chrome插件汇总

发表于 2018-12-23 分类于 工具汇总 阅读次数： Valine：

简介

Google Chrome 是一款快速、安全且免费的网络浏览器，能很好地满足新型网站对浏览器的要求。

这是Google浏览器官网的一段简单介绍。个人非常推崇Google浏览器，简洁、快速，Google浏览器最大的特色就是丰富的扩展程序，这些扩展程序更加成倍提升效率。用了Google浏览器一段时间，折腾了各种各样的扩展，深深体会到一个好的扩展程序的重要性。在此进行简单汇总。

以相关资料和个人经验，简单地做了一个脑图。最新浏览器绿色版下载：MyChrome_v3.8.1_x64

我们将插件分为以上12类进行介绍，仅根据个人经验和使用情况进行汇总，当然了，还有各种好用的插件。

插件介绍

自定义标签页

CaretTab

个人非常喜欢的一个新标签页插件，极简、极致。

Momentum

专注于一件事，并做到最好。

Infinity 新标签页

百万用户选择的新标签页，自由添加网站图标，云端高清壁纸，快速访问书签、天气、笔记、待办事项、扩展管理与历史记录。

(图片来自网络)

去广告插件

uBlock Origin

AdBlock

AdBlock Plus

之前查阅相关资料，最终选择了uBlock Origin，三种去广告插件都非常好用。

效率工具

OneTab

当我们打开过多的标签页时，点击OneTab 按钮，你当前所有打开的页面都会被关闭，整合呈现成为一个列表，显示在一个新打开的 OneTab 标签页中，并且再关闭浏览器之后仍然保存。

Evernote Web Clipper

大象官方的一个插件，可以直接把当前网页或者选中的文字直接保存的evernote，便于下次阅读和各终端同步，非常方便。

Imagus

浏览网页时经常有大量图片，有时候需要一个一个点击进行查看。通过这个插件，鼠标悬停即可自动预览图片，非常实用。

smartUp手势

通过鼠标手势更加方便地操作网页，对于网页很多链接，是从当前窗口打开的，而通过此插件，可以简单地拖拽网页即可从新标签页打开链接。

类似的还有国产的crxMouse Chrome Gestures

Vimium

像使用vim一样使用Chrome，Github地址：https://github.com/philc/vimium

使用Shift+/即可调出帮助文档，如可使用gg、Shift+g快速到达网站顶部和底部。

恢复关闭的标签页

通过点击此扩展，即可依次打开关闭的标签页，最后关闭的标签页当然最先打开。

LastPass

一键管理所有网站密码，生成所需要的密码并实现自动填充，非常方便且安全的密码管理扩展程序。

翻译插件

Google 翻译

好用简洁的翻译插件，堪称神器，可直接翻译网页，支持多种语言。

达达划词翻译

Github地址：https://github.com/waynecz/dadda-translate-crx
了解使用方法可点击进入Github，里面有详细的使用方法和截图，再次不在赘述。

Github相关

OctoTree

Github地址：https://github.com/ovity/octotree

GitHub Hovercard

Github地址：https://github.com/Justineo/github-hovercard ，里面有详细信息。

当你将鼠标停留在 GitHub 网站的用户头像或者仓库链接地址上时，会自动弹出一个悬浮框，带你提前预览基本信息，包括用户头像，仓库地址，Issue 链接，个人主页等。

Awesome Autocomplete for GitHub

GitHub 智能搜索插件。有点类似自动补全功能，实时监听输入的搜索关键字，自动列出相关内容。

专注

阅读模式

提供与Safari阅读模式功能一致的插件，浏览文章页时候可进入友好的阅读模式，并自定义阅读功能。

Forest

知名番茄钟应用 Forest 的 Chrome 插件版，Forest 支持在任意网页点击图标弹出番茄钟，你可以选择树苗和时间（最短不低于 25 分钟），也可以通过右上角的「＋」将当前网站快捷添加到「黑名单」。

可与Todoist配合使用。

Google相关

以下三款插件插件来自同一位大神之手，分别是Google的Gmail,Calendar和Google Drive的扩展程序，打开都是极简且功能强大的弹出框，参考如下。插件官方主页：点击进入

Checker Plus for Gmail

Checker Plus for Google Calendar

Checker Plus for Google Drive

神器类插件

Tampermonkey

Tampermonkey，又称“油猴”，通过安装各类脚本对网站进行定制。它能定制的不仅仅是网站的样式，还能实现更多更强大的功能，例如：

  • 直接下载百度网盘文件
  • 重新定制繁杂的微博页面
  • 去掉视频播放广告
  • 将网站默认的「二维码登录」改回「账号密码登录」
  • 绕过搜索引擎的跳转提示
  • 还原清新的小说阅读模式
  • 豆瓣和 IMDb 互相显示评分
  • ……

点击进入Google应用商店安装后，进入Greasy Fork即可下载安装各种脚本，或者点击我的另外一篇关于油猴脚本的汇总：自用油猴脚本推荐——让浏览器成为神器

脚本备份或同步
单击左键——管理面板——实用工具——云，可进行备份或同步。

Stylish-为任意网站自定义主题

通过在https://userstyles.org/ 上下载安装各种样式，获得更好的浏览体验。

比如通过安装BaiduX 极简百度+动态多栏+去下划线+毛玻璃导航栏+关键词高亮这样的一个样式，搜索百度时是这样的。

还有很多好用、好玩的样式，自行发现。

Pdf相关

Kami

PDF Viewer

直接将Pdf拖入到Chrome即可阅读，需先设置“允许访问文件网址”，选择扩展——右键——管理扩展程序。

这样即可愉快地使用Chrome阅读Pdf，具有目录导航，自动记忆阅读位置等功能。

代理插件

Proxy SwitchyOmega

轻松快捷地管理和切换多个代理设置,github地址：https://github.com/FelisCatus/SwitchyOmega，使用方法详见官方Wiki

网页截图

网页截图:注释&录屏

可截取网页可视部分，网页全部和截取选择部分，轻松实现网页长截图。

高效网页截图编辑插件

更加简单，易于使用。

扩展管理器

扩展管理器（Extension Manager）

一键管理所有扩展程序，一键禁用、启用插件。

类似的还有眼不见心不烦（新浪微博）,IG Helper: 下载 Instagram 图片和视频，但是这些可通过油猴脚本直接实现，后面我会对于常用的Tampermonkey（油猴)脚本进行汇总。
