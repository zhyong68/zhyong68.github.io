Linux系统虚拟内存的添加与删除

发表于 2018-09-04 分类于 Linux

虚拟内存的创建

登录到服务器，切换到root用户。

  1. 创建swap分区(512M,若没有/swap文件夹需创建)
    dd if=/dev/zero of=/swap/swap bs=1024 count=524288
  2. 创建Linux交换文件
    mkswap /swap/swap
  3. 激活/swap/swap交换文件
    swapon /swap/swap
  4. 查看结果
    free -m
  5. 加入到配置文件，使永久生效
    echo "/swap/swap swap swap defaults 0 0" >> /etc/fstab

虚拟内存的删除

  1. 停止swap分区
    swapoff /swap/swap
  2. 删除分区文件
    rm -rf /swap/swap
  3. 删除配置文件
    sed -i "/'\/swa\/swap swap swap defaults 0 0'//" /etc/fstab

记录以此备用。
