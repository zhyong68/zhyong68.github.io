autojump命令——Linux快速切换目录

发表于 2018-07-18 分类于 Linux

autojump目录切换神器

autojump是浏览文件系统的一种更快的方法。 它通过维护命令行中最常用的目录的数据库来工作，我们必须先访问目录，然后才能跳转到目录，根据使用频率切换，并且可以简写文件名等。

github地址：https://github.com/wting/autojump
官方Wiki：https://github.com/wting/autojump/wiki

之前写了mac如何安装autojump,觉得确实是一个神器，因此在Linux上安装，这里我们以Ubuntu/Debian系统为例。

安装

使用Ubuntu包管理apt进行安装：
sudo apt install autojump
如果没有找到autojump包，则运行：sudo apt update，然后再安装。

autojump支持的shell

通过安装autojump，在安装目录下找到支持的shell。
包括bash,fish,sh,zsh,软件默认安装在/usr/share/autojump.

使用方法

查看你默认使用的shell，运行cat $SHELL，然后再配置shell的配置文件中加入以下：
例如Ubuntu默认shell为bash，则在~/.bashrc中添加：
source /usr/share/autojump/autojump.bash
其它shell则相应更改即可。

例如打开文件夹即可使用j加目录关键字打开即可，直接按j可快速打开最常用的文件夹，使用起来非常方便。
