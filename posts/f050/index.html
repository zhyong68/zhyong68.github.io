树莓派 Raspberry Pi 3B+安装官方系统（一）

发表于 2020-06-20 分类于 树莓派

两年前购买的树莓派（可看当时我的记录树莓派 3B + 重装系统并配置 WiFi | YouForever）已经在角落里吃灰了好长时间，一度以为已经损坏，几天前试着拿出来折腾一下，竟然神奇地复活了，开心之余，也将树莓派重新利用了起来，现在，我已经离不开它了。

因此我想将树莓派写出一个系列，来说说我对它的使用。

一、 树莓派系统的选择及安装
二、 树莓派安装pi-hole过滤家中的广告
三、 树莓派连接蓝牙音箱，整点报时，定时语音播报

今天我们来进行系统的安装。

树莓派系统的安装

过了两年，我已经有自己的显示器了，想想当初没有显示器折腾树莓派，那叫一个辛苦。

首先，去官网：https://www.raspberrypi.org/downloads/ 下载安装镜像的文件，根据系统平台下载，直接安装即可，打开选择系统和SD卡，然后写入即可。

PS：我之前是从Github的一个仓库home-assistant/pi-gen: Tool used to create the Hassbian images下载的，镜像组件教老，问题较多，官方的实在太好用了！

完成后，将SD卡从读卡器取出，直接插入到树莓派（之前我是直接将读卡器插入到树莓派，结果运行中一堆问题，害的我以为树莓派坏了），接入显示器、键盘、鼠标，开机，即可看到成功进入树莓派的系统(图为vnc登录截的)。

到此，完成了树莓派Raspberry Pi官方系统的安装。

远程连接树莓派

SSH登录推荐使用PuTTY，默认用户名pi，密码为设置的密码，默认为raspberry。

VNC登录我使用的是VNC Viewer，非常不错。我使用的是网线连接（也推荐网线连接），因此我登录VNC后，直接点击右上角关闭Wifi。
