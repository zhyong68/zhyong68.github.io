【PC】电子书阅读软件推荐calibre

发表于 2019-02-23 分类于 Soft

需求

经常喜欢下载一些电子书，电子书常见的格式有.azw3，.epub，.mobi等，或者更加常见的.pdf格式。.pdf格式只需要一款pdf阅读器即可打开，但是对于.azw3，.epub，.mobi等格式，必须要特定的软件才能打开，因此非常需要一款PC端的电子书阅读器。这里推荐个人使用的三款电子书阅读软件，分别是Calibre（全平台 功能强大）、Book Bazaar Reader、Starrea（收费），体验都很不错。

电子书阅读软件

一款为跨平台且开源的阅读软件——calibre，另外一款为Windows的Metro应用——Book Bazaar Reader。

若想了解更多的相关软件，可查看知乎上的问答：PC端有什么好用的电子书阅读软件？ - 知乎

Calibre（推荐）

calibre为跨平台、开源的电子书阅读、管理软件，它可以查看、转换、编辑和编目所有主要电子书格式的电子书，支持Linux，Windows和macOS。Github上目前(2020年5月16日)已经发布了244个版本，开发非常火热。

下载地址：https://github.com/kovidgoyal/calibre/releases
若下载速度慢可留言，有需要我会尽量解决。

选择对应的操作系统进行下载，而且还贴心地为Windows提供了便携版——Calibre Portable。

打开软件后界面如下（Windows版):

使用calibre打开文件，可以看到几乎支持所有的电子书格式，

选中电子书，点击阅读即可，可以自动记忆阅读位置，字体大小等等都可控制，阅读体验很不错：

Calibre Web访问

并且此软件可提供Web访问，在“连接/共享”中启用内容服务，然后可内网访问,默认地址为 IP:8080。

Web访问的可玩性就比较高了，你可以使用Nas搭建（推荐）一个专门的Calibre Web服务，或者直接使用此软件，再配合内网穿透，可以在任何地方阅读家里设备的电子书。
janeczku/calibre-web: Web app for browsing, reading and downloading eBooks stored in a Calibre database

在浏览器打开地址，体验真的非常不错。

Book Bazaar Reader

此软件是Metro应用，需要在Mocrosoft Store中搜索Book Bazaar Reader进行下载，支持EPUB，MOBI，PDF, FB2，TXT格式图书。（只支持Windows 8 以后的系统）

相对于calibre来说，此软件应用范围有限——必须在微软商店下载，而且之前使用的过程中，打开某些电子书不能显示内容，而calibre可以正常阅读，兼容性不如前者。但是阅读体验也是不错的。

Starrea - EPUB 阅读与批注

Starrea - EPUB 阅读与批注 - Microsoft Store zh-CN
这是一款收费软件，价格不太清楚（10元左右），专注于电子书最流行的EPUB格式，用的不多，可自行体验。

其它

对于Android版电子书阅读器，个人觉得静读天下专业版很不错，网上有相关的破解版本。
