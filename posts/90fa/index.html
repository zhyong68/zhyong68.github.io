Anaconda简单介绍及使用

发表于 2018-10-29 分类于 Python3

Anaconda简介

Anaconda就是可以便捷获取包且对包能够进行管理，同时对环境可以统一管理的发行版本。Anaconda包含了conda、Python在内的超过180个科学包及其依赖项。

安装

以Windows为例：
官方下载：https://repo.anaconda.com/archive/Anaconda3-2019.07-Windows-x86_64.exe
安装过程很简单，一路next即可，注意最好将conda添加到环境变量中，更加方便使用。

常用命令

  • 升级包：conda upgrade --all
  • 创建环境：conda create -n xxx python=3
  • 环境列表：conda env list
  • 环境包列表：conda list
  • 激活环境：activate xxx
  • 安装第三方包：conda install requests
  • 卸载第三方包：conda remove requests
  • 查看当前环境包信息：conda list
  • 导出环境包信息：conda env export > environment.yaml
  • 导入环境包：conda env create -f environment.yaml
  • 复制环境：conda create --name <new_env_name> --clone <copied_env_name>
  • 删除环境：conda remove --name <env_name> --all

conda源修改

由于默认源下载速度较慢，我们使用清华源进行下载。
