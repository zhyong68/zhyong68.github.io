Rime——最好的输入法

发表于 2019-03-12 分类于 Soft 阅读次数： Valine：

最近迷上了五笔输入法，于是想找一款好用的五笔输入法软件，最终确定鼎鼎大名的Rime输入法——一款跨平台的开源输入法，支持Windows XP SP3, Windows 7+, macOS 10.7+, Linux (IBus, Fcitx) 。

简单地总结了一下使用汇总和配置五笔的步骤（输入法没有自带五笔）。

Rime输入法,各个平台下载地址：https://rime.im/download/
项目开源地址：https://github.com/rime

有意思的是，Rime输入法在各个平台下名称也是不同的：

  • rime/weasel: 【小狼毫】Rime for Windows
  • rime/squirrel: 【鼠鬚管】Rime for macOS
  • rime/ibus-rime: 【中州韻】Rime for Linux/IBus

另外，官方的五笔仓库：rime/rime-wubi: 【五筆字型】輸入方案。

由于在Linux系统上我没有安装使用Rime，故忽略Linux系统相关。

安装
Windows: 下载安装即可
MacOS: 使用brew安装：brew cask install squirrel

使用
输入时按F4键切换输入法方案。

配置五笔
Windows：

  1. 右键点击输入法中英文切换图标
  2. 在弹出的菜单中点击 “输入法设定”
  3. 点击 “获取更多方案”
  4. 在弹出的命令行窗口中粘贴 rime/rime-wubi 然后回车
  5. 按F4选择五笔86即可。
